This repository includes the code to process and analyze the data presented in the study:

"Plasma proteomics improves thrombosis prediction in cancer and implicates a targetable IL-17-driven endothelial activation pathway".

The data and code are organized in their respective directories.

In addition, the repository includes the `TOP_model` directory, which contains the pretrained TOP (Thrombosis Oncology Prediction) model, preprocessing normalization object, and scripts required to apply the TOP model for cancer-associated thrombosis (CAT) risk prediction on new patient datasets.
